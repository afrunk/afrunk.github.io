部署步骤&参考文章

  1. 安装node.js:下载链接

  2. 安装Git:下载链接

  3. 进入cdm输入以下代码验证是否安装成功

  4. 在github上创建并设置远程库
    新版本部署

    hexo d 没有反应，在配置文件中注意每个冒号后都有空格

    然后需要进入到git目录下输入邮箱和名字 然后就是再回来hexo d然后再是输入账号密码

  5. 安装hexo
    参考这篇文章：https://zhuanlan.zhihu.com/p/34379150

  6. 部署到github
    参考文章：https://zhuanlan.zhihu.com/p/34379150

  7. 主题
    https://jdhao.github.io/tags/ 该博客的主题：https://github.com/olOwOlo/hugo-theme-even。主题名：even.下载主题的方法：git clone https://github.com/olOwOlo/hugo-theme-even themes/even

还算不错的主题:

  • https://github.com/letiantian/huno 归档不太好看 而且还需要进行设置 但是主页太好看了 2分
  • https://github.com/hexojs/hexo-theme-light:还不错 就是旁边的布局部署非常好看，但是已经算不错了 3分
  • https://github.com/Haojen/hexo-theme-Claudia/blob/master/README-EN.md 十分不错的一个主题 4分
  • https://github.com/tufu9441/maupassant-hexo 最喜欢的主题了 之前就是为了她想转hexo的 https://www.haomwei.com/technology/maupassant-hexo.html 这个是中文文档 ，但是预览之后效果一般 有点难受 可能需要进行大量的修改 4.5
  • git clone https://github.com/ahonn/hexo-theme-even themes/even 目标主题之一 和上面的这个都是我最喜欢的 这个主题十分不错 部署起来了也还行 https://github.com/ahonn/hexo-theme-even/wiki/%E6%B7%BB%E5%8A%A0%E6%A0%87%E7%AD%BE%E9%A1%B5 文档页 就是你了 5分
  1. 发布文章
    进入到hexo目录下的source_posts（G:\BLOG\HEXO\source_posts），hexo new name 即可（名字如果有空格需要添加引号）。突然有一天出现 bash:hexo command not found问题，只需要将G:\BLOG\HEXO\node_modules\.bin添加到 PATH 中去即可。
  1. 发布文章之后index为空
    输入 npm ls 将未装插件装好 如 npm install eslint –save
    然后输入 npm audit fix

发现是主题的问题，even这么好看的主题为什么不能用呢？

  1. 插入图片
    直接在配置文件中将 post_asset_folder设置为 true 然后就会在目录下有一个同名文件夹用来存放图片了。

even 主题配置

  1. 在每个md文件头添加description: Blog Tools后面加描述文件即可

  2. 添加标签页：hexo new page tags 编辑 tags\index.html 设置 layout:tags即可，然后再主题配置文件（_config.yml）中添加,添加about页也是如此 hexo new page about,将布局设置为 layout:page,配置文件和语言文件都需要修改。

  3. 在G:\BLOG\HEXO\themes\even\languages的default文件中可以修改主页展示的五个栏目的内容

  4. 使用归档页布局作为首页：修改主题配置文件中 mode 字段可改变首页模式

    然后修改站点配置文件：首页显示全部文章，不分页
