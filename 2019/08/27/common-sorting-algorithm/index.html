1.3 Why C\C++

一般来说 Java 的执行比较慢，因此较常使用的是C或C++ 。考虑到C++的语法向下兼容C，并且C的输入输出语句比C++的要快很多，因此我们可以在主体上使用C语言的语法；而C++中有一些特性和功能非常好用（例如变量可以随时定义、拥有标准模板库STL等），因此在一定程度上我们可以混用部分C++的语法。

  • %md : 可以使得不足m位的int型变量以m位进行右对齐输出，其中高位用空格补齐，如果变量本身超过m位，则保持原样。

1.6 Math function

  • fabs(double x):对double型变量取绝对值
  • floor(double x) 和 ceil(double x)：向下和向上取整
  • pow(double r,double p):用于返回 rp ，要求r和p都是double型
  • sqrt(double x)：该函数用于返回 double 型变量的算数平方根
  • log(double x)：用于返回double型变量的以自然对数为底的对数：C语言没有对任意底数求对数的函数，只能通过换底公式来将不是以自然对数为底的对数转换为以e为底的对数，即 logab = logeb / logea
  • sin(double x )\cos(double x)和tan(double x)
  • round(double x)：适用于将double型变量x四舍五入，返回类型也是double类型，需进行取整。

2.1.1 如何使用STL sort 快排

为什么要使用STL的快速排序，因为有的时候冒泡、选择等算法的复杂度为 O(N^2)会超时，而快速排序的复杂度为 O (N log N)。

sort排序有两种形式，一个有三个参数，一个有两个参数。Sort的前两个参数是起始地址和中止地址。

如 sort(a,a+n)表示对a[0] , a[1] , a[2] … a[n-1] 排序 ，默认的是升序的。

如果我们要实现降序的话就要用到第三个参数，第三个参数是一个比较函数

冒泡排序的本质在于交换，即每次通过交换的方式把当前剩余元素的最大值移动到一端，而当剩余元素减少为0时，排序结束。
